# Telephasic – HTML & CSS UI Clone

## Introduction

Telephasic is a responsive **HTML & CSS UI Clone** of the sharp **Telephasic template by HTML5 UP**, focused on practicing modern frontend grid layouts, responsive design, and component styling using **pure HTML5 and CSS3 only**.

This project demonstrates a **versatile static website structure** with multiple page layouts such as **Homepage**, **Left Sidebar**, **Right Sidebar**, and **No Sidebar**, all connected through a clean navigation system and cohesive UI components.

---

## Screens Included

- **Homepage (Index)**: Hero introduction, dual feature rows, and a promotional call-to-action strip
- **Left Sidebar Page**: Layout featuring a sidebar on the left and main content on the right
- **Right Sidebar Page**: Layout featuring a sidebar on the right and main content on the left
- **No Sidebar Page**: Full-width content page for distraction-free reading
- **Navigation Bar**: Top header navigation with dropdown menu structure
- **Footer Section**: Contact form layout, social icons, and copyright info

---

## Features

### Homepage (Index)

- **Hero Banner**: High-impact introduction with a "Call to Action" button
- **Feature Grids**: Two distinct rows of 3-column feature blocks with images and text
- **Promo Section**: A dedicated full-width strip for special announcements or emphasis
- **Responsive Layout**: Adapts fluidly from desktop to mobile screens

---

### Left & Right Sidebar Pages

- **Split Layouts**: Demonstrates the use of grid systems to create unequal column widths (e.g., 4 columns vs 8 columns)
- **Sidebar Elements**: Styled sections for secondary links, images, or "blurb" content
- **Main Content**: Typography-focused area for articles and long-form text
- **Vertical Rhythm**: Consistent spacing between headers, paragraphs, and images

---

### No Sidebar Page

- **Full-Width Layout**: Utilizes the full container width (12 columns)
- **Centered Content**: Ideal for major announcements or simplified content presentation
- **Versatile Grid**: Shows the flexibility of the underlying CSS grid system

---

### Navigation

- Top-aligned navigation bar integrated into the header
- Dropdown menu structure (UI styling)
- Active state styling for current page indication
- Clean hover effects for improved usability

---

### Footer

- **Contact Form UI**: A styled layout for a name, email, and message input form
- **Social Icons**: Grouped brand icons (Font Awesome) for social media links
- **Copyright Area**: centered text links for legal and credits
- **Textured Background**: Uses overlay patterns for visual depth
